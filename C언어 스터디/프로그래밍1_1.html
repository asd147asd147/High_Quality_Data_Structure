    문제 내용

    ================= Question ==================

    아래 입출력 결과를 만들기 위해 주어진 조건을 만족하는 적당한 코드를 빈 칸에 넣어 프로그램을 완성하라․

    ---------------------------------------------

     

    ================ I/O Results ================

    1 2 3
    a + b == 3, c + d == 1
    계속하려면 아무 키나 누르십시오 ․ ․ ․
    +-------------------------------------------+
    3 -2
    9
    a + b == 1, c + d == 7
    계속하려면 아무 키나 누르십시오 ․ ․ ․

    ---------------------------------------------

     

    ================ Conditions =================

    • 정수형 변수 a, b, c, d를 차례로 선언한다․ 이때 d변수는 -2로 초기화하며 선언한다․
    • 키보드로부터 3개의 정수를 입력받아 순서대로 변수 a, b, c에 저장한다․
    • a와 b의 합과 c와 d의 합을 출력하되 입출력 결과와 동일하게 출력하도록 프로그램을 완성하라․
    • 주어진 소스 코드를 작성한 프로그래머의 의도를 이해하고 프로그램 요소 ❨❨타입, 상수, 변수, 함수 등❩❩를 최대한 활용하여 프로그램을 완성하라․

    ---------------------------------------------

     

    ================ Source Code ================

    #include <stdio․h>

    int main❨❨void❩❩
    {
        [############]
        int d = [####];

        [####]❨❨[####], [##], [##]❩❩;
        [####]❨❨[##], [##]❩❩;
        [####]❨❨[########################],
            [####], [####]❩❩;
        return 0;
    }

    ---------------------------------------------

     

    ================ Submission =================

    • 작성한 소스코드가 주어진 입출력 결과를 만족하면 시험 감독관에게 채점 받고, 통과하면 그 즉시 소스코드를 제출한다․
    • 소스코드는 반드시 압축하여 첨부파일로 제출하라․ 이때, 소스파일과 압축파일의 이름은 다음과 같이 한다․
    • 소스파일:
      • 형식❩❩ PE[문제번호2자리]_[학번]_[이름]․c
      예시❩❩ PE01_201712345_홍길동․c
    • 압축파일:
      • 형식❩❩ PE[문제번호2자리]_[학번]_[이름]․zip
      예시❩❩ PE01_201712345_홍길동․zip
