    문제 내용

    ================= Question ==================

    다음 프로그램은 아래 입출력 결과를 만들기 위해 작성되었다․ 소스 코드를 완성하기 위해 빈 칸에 들어갈 수 있는 코드를 모두 골라라․

    ---------------------------------------------

     

    ================ I/O Results ================

    1 10
    2 3
    2 3 4 6 8 9 10
    sum == 42
    계속하려면 아무 키나 누르십시오 ․ ․ ․
    +-------------------------------------------+
    20 100
    33 47
    33 47 66 94 99
    sum == 339
    계속하려면 아무 키나 누르십시오 ․ ․ ․

    ---------------------------------------------

     

    ================ Conditions =================

    • beg이상 end이하의 정수 중에 x의 배수이거나 y의 배수인 값을 출력하고 그 합을 구하여 출력한다․

    ---------------------------------------------

     

    ================ Source Code ================

    #include <stdio․h>

    int main❨❨void❩❩
    {
        int beg, end, i;
        int x, y, s=0;

        scanf❨❨"%d%d", &beg, &end❩❩;
        scanf❨❨"%d%d", &x, &y❩❩;
        for❨❨[####[a]####]❩❩
            if❨❨[####[b]####]❩❩ {
                printf❨❨"%d ", i❩❩;
                [####[c]####]
            }
        printf❨❨"\nsum == %d\n", s❩❩;
        return 0;
    }

    ---------------------------------------------

     

    =============== Choices for [a] ==============

    ① i=beg; i<end; ++i
    ② i=end; i>=beg; --i
    ③ i=beg; i<=end; ++i
    ④ i=beg; i<end+1; --i

    ---------------------------------------------

     

    =============== Choices for [b] ==============

    ⑤ x%i==0 || y%i==0
    ⑥ i%x==0 || i%y==0
    ⑦ !❨❨i%x!=0 && i%y!=0❩❩
    ⑧ x%i!=0 && y%i!=0

    ---------------------------------------------

     

    =============== Choices for [c] ==============

    ⑨ s += i;
    ⑩ sum = sum + i;
    ⑪ s = s + 1;
    ⑫ sum += sum + 1;

    ---------------------------------------------

     

    ================ Submission =================

    • 공백구분법: 답안 좌우에 공백 입력금지, 값❨❨또는 번호❩❩ 사이에 한 개의 공백만 사용
    • 오름차 및 순차기술: 각 보기 안에서는 값❨❨또는 번호❩❩를 오름차 정렬하여 기입하되, 각 보기들의 답안을 순차로 기술

    ---------------------------------------------

     

    문제 18

    정답
    총 1.00 점에서 1.00 점 할당

    문제 내용

    ================= Question ==================

    다음 프로그램은 아래 입출력 결과를 만들기 위해 빈칸에 들어가야 할 코드를 보기에서 골라 그 번호를 순서대로 기입하라․

    ---------------------------------------------

     

    ================ I/O Results ================

    2014
    not leap year
    계속하려면 아무 키나 누르십시오 ․ ․ ․
    +-------------------------------------------+
    2012
    leap year
    계속하려면 아무 키나 누르십시오 ․ ․ ․
    +-------------------------------------------+
    1300
    not leap year
    계속하려면 아무 키나 누르십시오 ․ ․ ․
    +-------------------------------------------+
    1200
    leap year
    계속하려면 아무 키나 누르십시오 ․ ․ ․

    ---------------------------------------------

     

    ================ Conditions =================

    • y가 4로 나눠지면 윤년이다․
    • 하지만, 바로 위의 조건을 만족하더라도 y가 100으로 나눠지면 윤년이 아니다․
    • 하지만, 바로 위의 조건을 만족하더라도 y가 400으로 나눠지면 윤년이다․
    • 위 3개의 조건을 C언어로 그대로 번역하여 3개의 if-else문으로 구현하라․

    ---------------------------------------------

     

    ================ Source Code ================

    #include <stdio․h>

    int main❨❨void❩❩
    {
        int y;

        scanf❨❨"%d",&y❩❩;

        if❨❨[########]❩❩
            if❨❨[########]❩❩
                if❨❨[########]❩❩
                    printf❨❨"leap year\n"❩❩;
                else
                    printf❨❨"not leap year\n"❩❩;
            else
                printf❨❨"leap year\n"❩❩;
        else
            printf❨❨"not leap year\n"❩❩;

        return 0;
    }

    ---------------------------------------------

     

    ================== Choices ==================

    ① y%4==0
    ② y%100==0
    ③ y%400==0
    ④ y%4!=0
    ⑤ y%100!=0
    ⑥ y%400!=0

    ---------------------------------------------

     

    ================ Submission =================

    • 공백구분법: 답안 좌우에 공백 입력금지, 값❨❨또는 번호❩❩ 사이에 한 개의 공백만 사용
    • 순차기술: 값❨❨또는 번호❩❩를 순차로 기입

    ---------------------------------------------
